Requirements

osCommerce Online Merchant can be installed on any web server that has PHP installed and has access to a database server. This includes shared servers, dedicated servers, cloud instances, and local installations running on Linux, Unix, BSD, Mac OS X, and Microsoft Windows operating systems.

Web Server

The web server must support PHP either as a module or allow execution of CGI scripts. For performance reasons FastCGI is recommended over CGI.

PHP

The minimum PHP version required is v5.3.0. It is recommended to use the latest PHP version on the web server.

The following PHP settings are recommended to be set in the php.ini configuration file:

PHP Setting Value
register_globals Off
magic_quotes_gpc Off
file_uploads On
session.auto_start Off
session.use_trans_sid Off

The following PHP extensions are mandatory:

Extension
PDO_MYSQL PDO MySQL database driver.

The following PHP extensions are recommended:

Extension
PDO_SQLITE PDO SQLite database driver.
Phar PHP Archive extension.
GD Image processing extension.
cURL Network communication library.
OpenSSL Encryption library.

MySQL Database Server

The minimum MySQL Server version required is v4.1.13+ or v5.0.7+. It is recommended to use the latest MySQL Server version on the database server.

The following MySQL Server storage engines are supported:

Storage Engine
MyISAM Standard storage engine.
InnoDB Transaction-safe storage engine.

osCommerce Online Merchant utilizes Foreign Key Constraints which is supported natively by InnoDB. A compatibility layer for the MyISAM storage engine exists where foreign key relationships are defined in the osc_fk_relationships database table.
