Changelog

osCommerce Online Merchant v2.4.0 (beta 1)

PHP Minimum Requirement

The PHP version minimum requirement is now v5.5.

The official list of PHP versions supported and in active development by the PHP Group can be found at:

http://php.net/supported-versions.php

Framework, Modularity, and Self-Contained Apps

A framework is being introduced to replace legacy functions and classes and to take advantage of newer PHP features.

The framework also allows for more modularity in the codebase and introduces the concept of self-contained Apps where extra functionality can be added without the need to edit core source code files.

The framework is located in the following directory:

Namespace Location 
OSC\OM
 
includes/OSC/OM

Super Globals and Session Variables

The usage of legacy PHP long variables has been replaced with PHP super globals and are no longer referenced as globals in functions.

Legacy Super Global
$HTTP_GET_VARS $_GET
$HTTP_POST_VARS $_POST
$HTTP_COOKIES_VARS $_COOKIES
$HTTP_FILES_VARS $_FILES
$HTTP_SERVER_VARS $_SERVER
(session varaibles) $_SESSION

Database

The legacy tep_db_* database functions have been replaced with the OSC\OM\Db class.

Bootstrap Frontend

Bootstrap is replacing jQuery UI as the frontend javascript framework. This has been done for the shop frontend and the setup routine.
