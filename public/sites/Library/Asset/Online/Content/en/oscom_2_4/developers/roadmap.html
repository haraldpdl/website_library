Roadmap

v2.4.0 (Beta 1)

OSC\OM Framework
Introduce a framework to replace legacy classes and functions with.
Bootstrap Frontend Framework
Replace the jQuery UI based Shop and Setup frontend with Bootstrap.
Self-Contained Apps
Introduce self-contained Apps with support for self-defined modules.

v2.4.1 (Beta 2)

INI Style Language Definitions
Move the language definitions to ini style files.
Legacy Cleanup
Migrate the following classes to the new framework:
  • Breadcrumb
  • Currencies
  • Language
  • Message Stack
  • Navigation History
  • Shopping Cart
SEO
Introduce SEO features.
Database Schema Comparison
Introduce a database schema comparison tool existing store owners can use as part of their upgrade.

v2.4.2 (Beta 3)

Legacy Cleanup
Migrate remaining classes and functions to the new framework.
Hook Action Calls
Finalize hook action calls throughout the codebase.

v2.4.3 (Beta 4)

Admin Bootstrap Frontend
Replace the jQuery UI based Admin frontend with Bootstrap.
Online Updates
Introduce an online update feature for both the core and installed Apps

v2.4.4 (Beta 5)

App Browser
Introduce an online App browser and installer.
Language Packs
Support community maintained language packs and provide a list of ready to be used languages in the setup routine.
